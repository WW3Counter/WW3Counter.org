# WW3 AI Probability Index 🎛️

A tongue-in-cheek AI-powered index that scans geopolitical news headlines and generates a "World War III Probability" percentage — powered by Claude AI.


## How It Works

1. **News Ingestion** — A scheduled script fetches the latest geopolitical headlines from RSS feeds (Reuters, Al Jazeera, BBC World, AP News, etc.)
2. **AI Analysis** — Headlines are sent to Claude AI with a prompt that scores current global tension on a 0–100 scale
3. **Index Generation** — The score is written to `docs/data/index.json` and committed to the repo
4. **Live Website** — GitHub Pages serves `docs/` which reads the JSON and displays the live counter

## Setup

### Prerequisites
- Node.js 18+
- An Anthropic API key

### Install
```bash
npm install
```

### Environment Variables
```bash
export ANTHROPIC_API_KEY="your-key-here"
```

### Run Manually
```bash
node scripts/update-index.js
```

### Automated Updates
The GitHub Action (`.github/workflows/update.yml`) runs every 6 hours to refresh the index automatically.
